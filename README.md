1. 找出基座坐标系下相机坐标系的值，当前状态下，是[-z,x,-y],
2. 相机坐标系需要从[x,y,z]转化为[-z,x,-y],故旋转矩阵是[0,0,-1],[1,0,0],[0,-1,0]，即RT*cam = base,
